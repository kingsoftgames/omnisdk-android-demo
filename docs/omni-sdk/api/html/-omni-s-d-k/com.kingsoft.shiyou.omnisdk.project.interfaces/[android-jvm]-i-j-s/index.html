JNI桥接方法：JS --> OmniSDK --> 游戏c/c++

  • JS 调用 SDK 提供的方法进而调用游戏本地c/c++库中的JNI桥接方法

  • 游戏最好提供可靠的验证方法或隔离接口，防止被恶意调用。

  • OmniSDK 只负责桥接、透传；不做任何验证。

JS 调用方法

function doFun() {
    // Js调用举例
    window.syAndroid.callNativeMethod("gameLib","methodName","methodSignature,"methodParam);
}

游戏CP方需实现如下头文件:

com_kingsoft_shiyou_omnisdk_project_web_OmniWebJni.h
#include <jni.h>
#ifndef _Included_com_kingsoft_shiyou_omnisdk_project_web_OmniWebJni
#define _Included_com_kingsoft_shiyou_omnisdk_project_web_OmniWebJni
#ifdef __cplusplus
extern "C" {
#endif

 // c/c++端和Java桥接的方法
 // 第一个jstring对应`methodName`: 方法名称标示
 // 第二个jstring对应`methodSignature`: 方法安全签名
 // 第三个jstring对应`methodParam`: 方法参数数据
 JNIEXPORT jstring JNICALL Java_com_kingsoft_shiyou_omnisdk_project_web_OmniWebJni_call
 (JNIEnv *, jclass, jstring, jstring, jstring);

#ifdef __cplusplus
}
#endif
#endif

JS 调用 SDK 进行分享

分享注意事件

  • 参考OmniSDK.Share

share-params-json 格式

{
 "platform": "facebook,line",
 "share_type": 0,
 "title": "I love this game",
 "content": "come to play with me now",
 "url": "https://developers.facebook.com",
 "image_type": 0,
 "image_uri": ""
}

参数含义及类型，参考OmniSDK.ShareParam

  • share_type: 0-LINK，1-GROUP_LINK，2-IMAGE

  • image_type: 0-NO，1-NETWORK，2-LOCAL，3-SCREENSHOT

  • image_uri: 如果传的是本地图片，请确保本地图片地址在Java方法，new File("local-path").exists() == true

  • platform

    • "facebook" 通过Facebook-App分享：需要接Facebook-plugin，否则通过系统分享界面分享

    • "system" 通过系统分享界面分享

    • "jp.naver.line.android" 指定 line 应用（无原生分享方法）；如应用不存在，通过系统分享界面分享；v1.0.9 开始可以用 "line" 代替。

    • 游戏可以指定特定应用包名，如果用户安装了，直接分享；如果应用不存在，通过系统分享界面分享。

    • "com.whatsapp" 指定 whatsapp 应用；如应用不存在，通过系统分享界面分享

JS 交互

function doFun() {
    window.syAndroid.share("{share-params-json}");
}
