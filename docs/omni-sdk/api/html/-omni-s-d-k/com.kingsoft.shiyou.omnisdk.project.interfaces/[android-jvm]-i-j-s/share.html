JS 调用 SDK 进行分享

分享注意事件

  • 参考OmniSDK.Share

share-params-json 格式

{
 "platform": "facebook,line",
 "share_type": 0,
 "title": "I love this game",
 "content": "come to play with me now",
 "url": "https://developers.facebook.com",
 "image_type": 0,
 "image_uri": ""
}

参数含义及类型，参考OmniSDK.ShareParam

  • share_type: 0-LINK，1-GROUP_LINK，2-IMAGE

  • image_type: 0-NO，1-NETWORK，2-LOCAL，3-SCREENSHOT

  • image_uri: 如果传的是本地图片，请确保本地图片地址在Java方法，new File("local-path").exists() == true

  • platform

    • "facebook" 通过Facebook-App分享：需要接Facebook-plugin，否则通过系统分享界面分享

    • "system" 通过系统分享界面分享

    • "jp.naver.line.android" 指定 line 应用（无原生分享方法）；如应用不存在，通过系统分享界面分享；v1.0.9 开始可以用 "line" 代替。

    • 游戏可以指定特定应用包名，如果用户安装了，直接分享；如果应用不存在，通过系统分享界面分享。

    • "com.whatsapp" 指定 whatsapp 应用；如应用不存在，通过系统分享界面分享

JS 交互

function doFun() {
    window.syAndroid.share("{share-params-json}");
}

Return

true 分享调用成功， false 分享调用失败
