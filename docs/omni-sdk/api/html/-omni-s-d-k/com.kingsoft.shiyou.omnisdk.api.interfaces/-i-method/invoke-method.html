invokeMethod

abstract fun invokeMethod(@NonNull() methodName: String, vararg objects: Any)
Content copied to clipboard

通过反射调用渠道独有方法

  1. 通过反射方法调用相应功能，使用前用isMethodSupported判断当前渠道是否支持

  2. 具体渠道支持的反射方法，查阅com.kingsoft.shiyou.omnisdk.api.interfaces.IChannelInvokeMethod

  3. 反射方法名部分可以通过 OmniConstant#invoke* 调用

  • 为保证一次接入，渠道独有接口，加上渠道判断后使用

  • 代码示例

    if ("transsion".equals(OmniSDK.getInstance().getChannelName())) {
        OmniSDK.getInstance().invokeMethod(OmniConstant.invoke*, param1...);
    }
