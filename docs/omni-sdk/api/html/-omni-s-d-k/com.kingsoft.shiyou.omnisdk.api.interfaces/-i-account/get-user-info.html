Content copied to clipboard

获取用户信息

  • 请在 login 登录成功后调用，未登录成功前返回 null。

UserInfo 参数

ParamNameDescription
uid平台 uid
cpUid平台 cpUid
token平台用户 token
showName用户名，空字符串此渠道无用户名
certification实名制及是否成年状态：-1 渠道实现防沉迷；0 未实名制；1 未成年；2 已成年；其他值：该应用及渠道无实名信息。
type账号类型 AccountType
isRelated是否为游客账号
ext扩展参数，建议使用 OmniUtils.getValue 获取对应值

Return

null || InfoUserJsonString
