获取用户信息

获取用户信息

请在 accountLogin 登录成功后调用，未登录成功前返回空。

用户信息 UserInfoJson 结构体

名称描述
cpUidOmniSDK 返回给游戏的账号ID
tokenOmniSDK 返回给游戏的token，用于静默登录
accountType登录的账号类型：AccountType；
游客绑定社交帐号后显示为该社交帐号的类型。
userName用户名：渠道可能返回空字符串
certification（可能为空字符串）实名制及是否成年状态：-1：渠道实现防沉迷；0：未实名制；1：未成年；2：已成年；其他值：该应用及渠道无实名信息。
isGuest是否为游客账号
verifySign海外：无，不需要；
国内：账号的验证Sign值；安全上禁止本地存储。
verifyTimestamp海外：无，不需要；
国内：Sign值的Unix时间戳，单位为秒；安全上禁止本地存储。
ext扩展参数

Return

null 或 UserInfoJson
