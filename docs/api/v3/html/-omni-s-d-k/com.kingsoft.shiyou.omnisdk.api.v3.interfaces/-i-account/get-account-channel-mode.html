获取当前渠道账号体系的模式

获取当前渠道账号体系的模式

描述

  • 仅在自定义登录UI界面时使用。

  • OmniSDK 为聚合类型 SDK，需要处理不同渠道对账号类型是否显示的不同要求，以保证各渠道平台的审核发布。

自定义登录UI界面

  • 游戏在自定义UI界面及逻辑时，先调用本接口查询渠道账号体系的模式；

  • 根据返回的结果来处理：账号的显示、或业务逻辑处理。

@return描述
1渠道无账号，即允许 OmniSDK 支持的所有账号类型。
2渠道有账号且禁止其他账号显示，即只允许渠道账号显示。
3渠道有账号但允许其他账号显示，即必须包含渠道账号的显示。
