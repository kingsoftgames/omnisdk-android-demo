被渠道主动踢出登录状态

被渠道主动踢出登录状态

可能的场景触发

  1. 用户在渠道的账号界面执行登出功能后触发。

  2. 渠道防沉迷机制触发：此时 forceLogout 为 true。

  3. 其他的渠道未知原因。

部分要求强杀游戏进程的渠道列表

  • 目前渠道有：nubia

  • 当 forceLogout 为 true 时，渠道要求强制杀掉游戏应用，只是回到登录界面，渠道会检查进程状态，造成显示异常。

  • 请通过 OmniSDK.getInstance().getChannelName() 判断渠道，保存数据后，按渠道要求处理。

  • 其他渠道可以返回登录界面，具体以渠道的最新审核政策为准。

Parameters

forceLogout

是否被强制踢出登录状态
