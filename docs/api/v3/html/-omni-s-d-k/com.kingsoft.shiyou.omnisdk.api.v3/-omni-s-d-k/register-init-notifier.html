描述

  1. 根据隐私政策或审核政策，游戏需要在用户同意相关隐私协议后，进行后续的操作。

  2. 根据不同国家政策、渠道政策要求，OmniSDK 提供初始化监听接口 registerInitNotifier，游戏需要在收到初始化成功回调后，再进行后续操作，以免影响政策或审核。
