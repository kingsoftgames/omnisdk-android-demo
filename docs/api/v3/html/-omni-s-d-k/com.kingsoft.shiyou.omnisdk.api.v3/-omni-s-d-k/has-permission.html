是否拥有某个权限或权限组

是否拥有某个权限或权限组

描述

  1. 判断游戏是否拥有某个权限或权限组；

  2. 游戏通过 android.Manifest 获取权限或权限组的字符串(Constant Value)；

  3. 一次只检查一个权限或权限组。

  4. 可以通过 android.Manifest Android 线上文档 查看权限字符串(Constant Value)

示例

// 检查存储 权限组
OmniSDK.getInstance().hasPermission(android.Manifest.permission_group.STORAGE); // 通过代码引用
OmniSDK.getInstance().hasPermission("android.permission-group.STORAGE"); // 通过 Constant Value 使用

// 检查存储的单个权限
OmniSDK.getInstance().hasPermission(android.Manifest.permission.WRITE_EXTERNAL_STORAGE); // 通过代码引用
OmniSDK.getInstance().hasPermission("android.permission.WRITE_EXTERNAL_STORAGE"); // 通过 Constant Value 使用
