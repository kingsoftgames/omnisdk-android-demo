通过反射调用渠道独有方法

通过反射调用渠道独有方法

描述

  1. 使用前用 isMethodSupported 判断当前渠道是否支持

  2. 具体渠道支持的反射方法，查阅 com.kingsoft.shiyou.omnisdk.api.interfaces.IChannelInvokeMethod

methodJson 结构体

字段类型描述
methodNameString反射的方法名
paramsString1. 多个参数值以 ; 符号分割。
2. 无参数时传空字符串。
3. List 要转成 JSON-String，注意符号转义

methodJson 示例

{
  "methodName": "methodName",
  "params": "p1;p2;{\"p3\":\"methodName\"};p4"
}
