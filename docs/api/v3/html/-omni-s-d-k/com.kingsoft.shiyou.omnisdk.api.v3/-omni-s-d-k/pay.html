支付

描述

  • 调用支付接口前，用户必需先成功登录账号 IAccount.accountLogin。

  • 不要使用特殊字符，包括 # " & / ? $ ^ *:) \ < , = 回车 换行等，否则部分渠道无法拉起支付。

商品信息 productJson 结构体

字段类型长度描述必须
skuTypeInt1商品类型：0 - 普通内购；1 - 订阅类型Y
gameTradeNoString64游戏订单ID，不可重复的字符串Y
productIdString64商品ID，与所有后台商品ID保持一致，否则渠道无法拉起支付Y
productNameString64商品名称；不能包含特殊字符，包括 # " & / ? $ ^ *:) \ < , = 回车 换行，否则部分渠道无法拉起支付Y
productDescString128商品描述；不能包含特殊字符，包括 # " & / ? $ ^ *:) \ < , = 回车 换行，否则部分渠道无法拉起支付Y
productPriceDouble10商品单价；单位：元、美元，比如9.99美元，0.99港币等Y
productNumString10商品数量Y
totalAmountDouble10商品总额；单位：元、美元Y
payAmountDouble10实际支付总额；单位：元、美元Y
currencyString64实际支付的国际标准货币代码，比如CNY(人民币)/USD(美元)Y
serverIdString32服ID：必须为纯数字，且不能超过2147483647（应用宝渠道要求）Y
zoneIdString32区ID国内-Y，海外-N
roleIdString32角色IDY
roleNameString32角色名称国内-Y，海外-N
roleLevelString32角色等级国内-Y，海外-N
roleVipLevelString32角色的VIP等级国内-Y，海外-N
gameCallbackUrlString128支付回调地址，如果为空字符串，则使用后台配置的回调地址Y
extJsonString2000扩展参数，JSON 格式；无传空字符串Y
