uploadCrash

Integer uploadCrash(Context context, Map<String, Object> params)
Content copied to clipboard

crash 日志上传Omni；debug 模式下不上传。

  • 建议游戏，在用户主动反馈Bug时，同时调用本接口上传 omni-crash 目录给 omnisdk 服务器

  • 本接口只会收集上传sdcard/Android/data/packageName/files/crash_omni和data/data/packageName/files/crash_omni 目录下的crash日志。

  • 反馈问题时，请带上crash目录的日志。

Return

0: 上传成功；-1或其他: 上传失败

Since

1.0.5 服务器接口未上线，暂时无法上传
