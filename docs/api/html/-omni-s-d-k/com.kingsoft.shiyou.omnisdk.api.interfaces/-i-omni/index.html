登录方法
指定登录时，key值为loginType

  • 0，表示不指定特定登录类型，直接使用默认登录界面；

  • 非0，表示指定特定登录类型，游戏使用自身的登录选项UI界面，这种方式游戏需要额外自己保存、更新登录信息；

  • 1 为游客登录

  • 2 为邮箱账号

  • 3 为Facebook账号

  • 4 为Google账号
