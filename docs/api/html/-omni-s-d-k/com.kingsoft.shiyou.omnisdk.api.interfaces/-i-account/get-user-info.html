getUserInfo

@Nullable()
abstract String getUserInfo()
Content copied to clipboard

获取用户信息，未登录时返回空值

  • InfoUserJsonString

      {"uid":"","token":"","showName":"","channelId":"","certification":"","isRelated":"","ext":"extJsonString"}

  • extJsonString

      {"imageUrl":"","gender":"","nickName":"","width":"","height":""}

Return

Nullable || InfoUserJsonString

Since

1.0.1

Parameters

uid

平台 uid

token

平台用户 token

showName

用户名，空字符串此渠道无用户名

channelId

渠道 ID

certification

实名制及是否成年状态：-1 渠道实现防沉迷；0 未实名制；1 未成年；2 已成年；其他值：该应用及渠道无实名信息。

isRelated

true 表示为非纯游客，false 表示为纯游客

ext

扩展参数：目前只有 facebook 登录和 Google 登录有额外的渠道信息参数

imageUrl

平台头像链接

width

头像的宽度（如有，无则是 0 ）

height

头像的高度（如有，无则是 0 ）

gender

性别，返回 female or male ；如是其他信息则表示用户未设置性别

nickName

第三方渠道昵称
