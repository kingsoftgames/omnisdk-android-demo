分享：单个平台分享

  • platform is "facebook" 通过Facebook-App分享

  • platform is "system" 通过系统分享界面分享

  • platform is "com.whatsapp" 指定系统分享应用；如应用不存在，通过系统分享界面分享

Since

1.0 支持 facebook、system、指定包名
